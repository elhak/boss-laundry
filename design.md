## Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Primary Cyan | `#00BCD4` | Headers, buttons, accents |
| Primary Dark | `#0097A7` | Hover states, emphasis |
| Primary Light | `#B2EBF2` | Backgrounds, highlights |
| Black | `#1A1A1A` | Text, logo elements |
| Dark Gray | `#333333` | Body text |

---

## Typography

- **Headings:** Poppins (600, 700, 800) - Modern, friendly, approachable
- **Body:** Inter (400, 500, 600) - Clean, highly readable
- **Prices:** Roboto Mono (500) - For pricing tables, looks professional

---

## Section Structure

1. **Navbar** - Logo + Nav links + CTA button
2. **Hero** - Big headline + tagline + CTA + banner image
3. **Services** - 6 service cards with icons
4. **Pricing** - Tabbed pricing tables (Kiloan, Satuan, Express, etc.)
5. **Why Us** - Benefits (Gratis Antar Jemput, Express, etc.)
6. **How It Works** - 3 steps visual
7. **Contact/CTA** - WhatsApp button + Location
8. **Footer** - Links, social, copyright

---

## Visual Style

- **Mood:** Clean, fresh, professional, trustworthy
- **Shapes:** Rounded corners (16px-24px), soft shadows
- **Icons:** Lucide icons or Heroicons (outlined)
- **Imagery:** Bubble/water motifs, clean fabric textures
- **Animations:** Subtle fade-ins, hover lift effects

---

## Key Design Decisions

1. **Hero:** Large banner image with overlay gradient, prominent CTA button
2. **Pricing:** Tabbed interface to avoid overwhelming users
3. **WhatsApp CTA:** Sticky floating button for easy contact
4. **Trust signals:** "Gratis Antar Jemput" badge prominently displayed
5. **Mobile-first:** Single column on mobile, grid on desktop
